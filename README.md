\# Projet CL-Drive



Ce dépôt contient les travaux réalisés dans le cadre du TP sur les interfaces cerveau-machine et l'analyse des signaux EEG.



Le projet s'appuie sur le jeu de données CL-Drive, dont l'objectif est de mesurer la charge cognitive d'un conducteur à partir de signaux physiologiques.



\## Contenu du dépôt



\- TD1 : étude et compréhension de l'article scientifique ;

\- TD2 : exploration et préparation des données ;

\- TD3 : extraction des caractéristiques EEG ;

\- TD4 : calcul et vérification des features ;

\- TP : réalisation complète du pipeline de classification EEG.



Le notebook du TP contient les explications, le code, les résultats obtenus ainsi qu'une discussion des performances conformément aux consignes du cours.
